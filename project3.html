Terrible Tweeters

View on GitHub

Terrible Tweeters is a 2D physics-based puzzle game inspired by Angry Birds, recreated from Jason Weimannâ€™s Unity tutorial. Players launch birds to topple structures and defeat enemies using a drag-and-release slingshot mechanic, with destructible environments and satisfying physics interactions.

This custom iteration builds upon the original project by adding refined mechanics and a full game loop. The game now features custom art and sound effects and new handcrafted levels and a title screen and a custom win screen with "Thanks for Playing" message and restart options.

  • Slingshot launching system using mouse input and Rigidbody2D physics
  • Projectile trajectory prediction with line renderers
  • Destructible structures built from stacked colliders
  • Enemy knockout mechanics through collision or destabilization
  • Level reset and retry logic with basic win conditions
